# KIJINKAI 구매대행서비스

## 1.프로젝트 개요

### 1. 개발 배경
- 계기: 친구가 진행하던 사업(일본 상품을 칠레로 대량 판매)에 대한 이야기를 듣고 개발을 시작했습니다.
- 문제점: 기존에는 모든 과정을 수동으로 시트(Spreadsheet)에 기입하며 관리하는 비효율적인 방식이었습니다. 특히, 칠레로의 배송 과정이 수기 작업의 어려움을 보여주는 주요 사례였습니다.
- 해결 과제: 이 수동 작업을 시스템으로 전환하여 대량 주문 처리에 대한 부담을 줄이고 정확성을 높이는 것입니다.

### 2.프로젝트 목적 및 목표
- 목적: 일본에서 판매되는 상품을 한국,칠레에서 비즈니스를 운영하는 유저(사업자)들을 대상으로 대량 판매를 진행하기 위해 개발되었습니다.

### 3.개발환경

- 언어 : Java
- 프레임워크 : Spring Boot
- 데이터베이스 : MySql
- ORM: Jpa


### 4.핵심 기능

- 상품 정보 수집: 유저로부터 상품 URL 및 상품 개수 정보를 입력받습니다.
- 구매/재고 체크: 입력받은 정보를 바탕으로 구매 가능 여부를 체크합니다.
- 해외 배송: 구매 및 체크가 완료된 상품을 최종적으로 해당 나라(한국, 칠레)의 유저에게 배송합니다.
